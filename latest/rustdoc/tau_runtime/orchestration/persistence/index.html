JSONL run-log writer subscribed to the TraceStream.

Writes <scope>/.tau/runs/<run-id>.jsonl. Each line is either a TraceEvent or a TaskMutation projection (for replay-ability). Crash-safe: fsync after each line; replay tolerates trailing partial.
