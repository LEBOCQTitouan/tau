Module tracing_layer 

Source
Expand description

tracing-subscriber JSON layer that writes structured events to stderr. The host (in tau-runtime::plugin_host) reads each line, decodes the JSON, and re-emits as a tracing::Event on target = "plugin::<plugin_name>".

Internals delegate to tau_observe::install so all tau crates share one subscriber-init code path.
