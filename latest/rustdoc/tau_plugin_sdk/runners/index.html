Per-port plugin runners.

Each runner drives the full plugin lifecycle for one [tau_domain::PortKind]: install the SDK tracing layer, run the handshake, dispatch incoming frames to the plugin’s port-specific methods, and exit cleanly on meta.shutdown or stdin EOF.

See docs/superpowers/specs/2026-04-28-plugin-loading-design.md §5.2.
