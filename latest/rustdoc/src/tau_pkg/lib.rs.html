5//! Tau package manager. Resolves, installs, and verifies extension
6//! packages declared by users via `tau install`.
7//!
8//! tau-pkg implements:
9//!
10//! - **Scope detection** (G8): walks up from the cwd looking for a
11//!   `.tau/` directory, falls back to global scope (`~/.tau`).
12//! - **Manifest parsing** (G14): reads and structurally validates
13//!   `tau.toml` files from disk via `tau_domain::UncheckedManifest`.
14//! - **Install / uninstall**: shells out to `git clone`, materializes
15//!   the package source tree, updates the lockfile.
16//! - **Lockfile**: versioned TOML at `<project>/tau-lock.toml`
17//!   (committed) or `~/.tau/tau-lock.toml` (local).
18//!
19//! See `docs/decisions/0004-tau-pkg.md` for the design rationale.
