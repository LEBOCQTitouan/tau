ShellPlugin — Tool impl for the shell plugin.

Runs allow-listed subprocesses with wall-clock timeout + output capping. The agent’s grant (received via SessionContext.granted_capabilities) carries the ProcessCapability::Spawn.commands allow-list that constrains which command names are admissible.
