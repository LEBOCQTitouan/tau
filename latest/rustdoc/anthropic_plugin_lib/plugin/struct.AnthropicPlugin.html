Anthropic Claude (Messages API) plugin.

Constructed via Configure::from_config from an AnthropicConfig. Holds an AnthropicClient preconfigured with the resolved API key, base URL, API-version header, retry policy, and a reqwest::Client carrying the per-request HTTP timeout.
