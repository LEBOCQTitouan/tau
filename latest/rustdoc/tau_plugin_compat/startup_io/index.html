Per-plugin startup-IO path helpers for Layer 4 tests.

The tau-sandbox-native::light::BASELINE_SYSTEM_READ_PATHS runtime constant covers paths that every Rust binary needs (libc/dyld bootstrap, /proc/self, /sys/fs/cgroup, etc.). This module supplies the long-tail of plugin-specific paths beyond that baseline — for example, an HTTP plugin that reads a distribution-specific TLS root cert bundle that isn’t covered by the /etc baseline entry.

Tests in tau-plugin-compat/tests/layer4_native.rs call startup_io_paths_for(plugin_bin) and add the returned paths as an additional cap_fs_read entry in the test’s SandboxPlan.
