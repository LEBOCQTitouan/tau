Module cassette

Expand description

In-process HTTP cassette replayer for plugin integration tests.

Loads YAML cassettes describing recorded request/response pairs and serves them in-order from a tokio::net::TcpListener. Captures the request body (and arbitrary other headers) into a Vec<RecordedRequest> so tests can assert on what the plugin sent.

Originated in crates/tau-plugins/anthropic/tests/common/cassette.rs; lifted here as the rule-of-three refactor when ollama and openai became consumers.
