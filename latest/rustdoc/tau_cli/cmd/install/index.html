tau install — install a package from a git URL.

Per spec §3.13 install row:

  • Parses the URL into a PackageSource.
  • Resolves the active Scope (project or global).
  • For --dry-run: clones the source into a tempfile::TempDir, parses the manifest, prints a preview of what would be installed, and writes nothing to the scope.
  • Otherwise: delegates to tau_pkg::install_with_options and prints either a one-line summary or, when --json is set, a structured payload.

Errors propagate via anyhow::Error and exit 2 through the top-level dispatch in crate::run_main.
