tau uninstall — remove an installed package from the active scope.

Per spec §4:

  • Resolves the active Scope (project or global).
  • Parses the package name and optional version.
  • Delegates to tau_pkg::uninstall.
  • Prints either a human-readable summary with a remediation hint (§4.2) or, when --json is set, four per-line JSON events (§4.3).

Errors propagate via anyhow::Error and exit 2 through the top-level dispatch in crate::run_main.
