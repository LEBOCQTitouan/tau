tau plugin {describe, run, protocol decode} subcommands.

Spec §9 (debug tier): these handlers expose the plugin protocol to human operators for inspection, REPL-style probing, and post-hoc analysis of recorded sessions.

  • describe resolves a plugin from the lockfile, drives one handshake, prints the advertised metadata + per-method schemas, then shuts the plugin down cleanly.
  • run spawns an arbitrary plugin binary and drives either an interactive REPL (<method> <json-args> per line) or a scripted JSONL input file. Standalone — no agent, no kernel.
  • protocol_decode reads a JSONL recording produced by --record-protocol and emits a human-readable transcript with filter/time-range/JSON projection options.
