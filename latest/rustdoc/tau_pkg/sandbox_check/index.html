Module sandbox_check 

Source
Expand description

Layer 2 install-time cross-check: spawn the plugin binary, perform the meta.handshake RPC, and compare the capabilities the binary claims (via tool.describe_capabilities) against what the manifest declares.

For tool-port plugins the check is bidirectional — binary-claims-extra and manifest-declares-unused both hard-fail. For LLM-backend and storage plugins the binary-side wire mechanism is deferred per ADR-0016 Decision 1; those paths return the manifest’s capability list verbatim.

The public entry point is cross_check_plugin_capabilities.
