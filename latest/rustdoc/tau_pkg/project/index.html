Project-level tau.toml parser, validator, and agent resolution.

Lifted from tau-cli::config 2026-05-17 so that other binaries (notably tau-app::serve) can resolve agents without depending on the tau-cli crate.

Distinct from the package-level tau.toml defined in tau-domain (per ADR-0002); see ADR-0007 for the Cargo [package] vs [workspace] precedent that justifies the shared filename.
