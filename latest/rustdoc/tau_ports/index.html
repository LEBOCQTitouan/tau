Crate tau_ports 

Source
Expand description

Port (trait) definitions for tau’s hexagonal architecture. Adapters in tau-infra implement these traits.

tau-ports defines four trait families:

  • llm::LlmBackend — LLM provider plugins (kind = "llm-backend").
  • tool::Tool — tool plugins (kind = "tool").
  • storage::Storage — storage plugins (kind = "storage").
  • sandbox::Sandbox — sandbox adapters; probe-based adapter selection for OS-native and container sandboxing.

See docs/decisions/0003-tau-ports.md for the design rationale.
