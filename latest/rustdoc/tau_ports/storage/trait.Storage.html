Expand description

Trait implemented by kind = "storage" plugins.

v0.1 surface is KV-only. Per G8, the namespace carries scope (e.g., global / project / agent-instance); tau-runtime composes namespaces and plugins consume them opaquely.

Send + Sync so the runtime can store impls in a multi-task plugin registry.

Delete the key. Returns true if a key was deleted, false if it wasn’t present.

Source

async fn list( &self, namespace: &Namespace, prefix: &str, ) -> Result<Vec<Key>, StorageError>

List all keys under namespace whose names begin with prefix. Use empty string "" to list all keys in the namespace. Order is plugin-defined — callers must not rely on alphabetical.
