Identifier newtypes used across the tau-domain surface.

PackageName and AgentId are validating ASCII kebab-case identifiers. AgentInstanceId and MessageId are UUID v7-based opaque identifiers.

Structs§

AgentId
An agent identifier. ASCII kebab-case, must start with a lowercase letter, 1..=64 characters, character set [a-z0-9-].
AgentInstanceId
A runtime instance identifier for a spawned agent. UUID v7 (monotonic, time-ordered). Two instances of the same AgentDefinition share an AgentId but differ in AgentInstanceId.
MessageId
A message identifier. UUID v7 (monotonic, time-ordered). Acts as the reply target for Message.parent_id.
PackageName
A package name. ASCII kebab-case, must start with a lowercase letter, 1..=64 characters, character set [a-z0-9-].
