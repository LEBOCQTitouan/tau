Capability declarations attached to a package manifest.

Hierarchical typed enum: top-level by namespace (Filesystem/Network/Process/Agent/Custom), per-namespace verb enums underneath. Variant-level #[non_exhaustive] permits additive field evolution.

Wire format per ADR-0002: manifest TOML uses flat dot-namespaced kind = "fs.read" form. The custom Deserialize impl on Capability maps it onto the variant tree.
